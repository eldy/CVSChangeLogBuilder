Description:
cvschangelogbuilder is an Perl utility to generate ChangeLogs for a project hosted on a CVS server. The main goal is to provide a better output that the 'cvs log' command, to permit to choose the sorting criteria, to add the status of the change (added,removed,modified) and to allow "differential" changelogs, between 2 versions, even when they are not in the main branch.
